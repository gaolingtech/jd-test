#### AI 应用研发工程师（AI文档方向）

**岗位职责**

负责公司Word文档AI处理与生成平台的设计与开发，包括文档识别、文档自动生成、文本摘要、问答系统、检索增强生成（RAG）等模块；
参与大模型（LLM、多模态模型）集成与工程化应用，设计高性能任务调度、异步处理及多模型调用架构；
设计并优化分布式系统、高并发系统和微服务架构，确保 AI 文档服务稳定性和高可用性；
负责数据库设计、缓存优化、消息队列及中间件使用，支持文档处理任务的大规模并发执行；
编写高质量、可维护的代码，参与系统性能调优和疑难问题排查；
与前端、产品及数据团队紧密合作，将 AI 文档处理能力落地到实际业务场景。

**任职要求**

计算机软件或相关专业本科及以上学历，1 年以上研发经验，具备扎实的计算机基础（数据结构、操作系统、计算机网络等）；
精通 Java 或 Python 编程，熟悉 IO、多线程/协程、集合/数据结构，理解内存管理和性能优化；
熟练使用后端开发框架（如 Spring Boot / Spring Cloud / MyBatis / JPA / FastAPI / Django / Flask 等），理解其实现原理；
具备分布式系统或高并发系统开发经验，熟悉微服务架构及常用中间件（Redis、Kafka、RabbitMQ、RocketMQ 等）；
熟悉关系型数据库（MySQL / PostgreSQL / Oracle），有数据库优化、分库分表经验；
熟练使用 Linux / Unix 系统，能够快速编写 Shell 或 Python 脚本进行自动化任务；
具备系统调试、性能优化能力，对疑难技术问题具备较强排查能力；
具备自驱力、求知欲和团队协作精神，能够推动项目落地。

**加分项**

有 AI 文档处理经验，如文档自动化处理、文档摘要、知识图谱或信息抽取；
熟悉向量检索、Embedding 技术及向量数据库（Milvus / FAISS / Elasticsearch）；
熟悉 GPU 推理及深度学习框架（PyTorch / TensorFlow / CUDA）；
对算法优化、文本处理和数据挖掘有兴趣，能根据业务需求改进算法。
